# Dots in streets
## Making THREE.js points follow svg paths with light effects

### To get the project to work:
Install Parcel using Yarn or npm:
```
Yarn:
yarn global add parcel-bundler
```

```
npm:
npm install -g parcel-bundler
```

Run Parcel:
```
parcel index.html
```
### Technologies used:
- JavaScript
- Three.js
- WebGL (shader)
